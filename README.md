# SDA Hymnal+
Este himnario está siendo desarrollado para cumplir con el requisito específico de poder ser utilizado en 2 idiomas simultáneamente.

Desarrollado para las plataformas de sobremesa (Windows, Mac, Linux) Usando:
- [Svelte](https://svelte.dev/)
- [Pocketbase](https://pocketbase.io/)
- [Tauri](https://tauri.app/)
